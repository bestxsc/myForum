# myForum
### 简易的在线公告板Java Web应用
#### Genesis V 0.01
前端基于 Bootstrap 框架和Bootstrap-table、summernote组件开发，以JDBC操作mySQL数据库、Redis作为Key-Value缓存基于JSP技术开发后端的初学者练手小应用。
初学Java Web时起于一时兴起，历时一周的课余时间初步完成一个及其简单功能的在线公告板应用程序。
